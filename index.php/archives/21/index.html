文献阅读记录（四）

dhui •  2023-10-28  • 0 评论  • 55 阅读

用于记录阅读过的文献，总结文章要点，便于索引。

主要文章采用二级标题，相关文章采用三级标题。

HRBF-Fusion Accurate 3D Reconstruction from RGB-D Data Using On-the-fly Implicits

三维点云重建，使用 HRBF 隐式曲面的方法。

阅读该文章主要学会的是 HRBF 隐式曲面表示和 Linux 代码编译流程。计算机视觉相关的几个步骤没有看懂。

此论文代码已编译完成但未测试使用，已搁置。

Hermite Radial Basis Functions Implicits

HRBF 曲面的一般形式，给定点集以及相应的法向量，通过解线性方程组求解系数，得到隐式曲面表示。

Future Work（想法）：加入PDE参数做动画

A closed-form formulation of HRBFbased surface reconstruction by approximate solution.

HRBF 曲面的近似解，不需要解线性方程组，可用于实时计算。

Geometric Optimisation Via Spectral Shifting

使用 Spectral Shifting 进行几何优化的方法，重点是可以恢复三角网格翻转。

SVD 分解+正则化，推广相关能量函数。

不保证 Bounded Distortion ，没有理论保证能量函数的全局极小点是 injective map 。

学到了 Just Accepted 的意思。文章比前一篇 CV 方向的阅读难度低很多，关联文章也看懂了。

Lifting Simplices to Find Injectivity (TLC)

单射 (Injective Mapping) 的性质。

该论文方法可以恢复三角形翻转，通过引入辅助平面，将空间维度提高的方式。

有理论保证存在全局极小值，但因为能量函数非凸，很难落到该处。

处理 Fixed Boundary 问题。

Foldover-free maps in 50 lines of code (FFM)

其中最早的一篇解决翻转问题的方法，简单易懂，有 presentation 视频。

通过正则化能量函数表达式，使得优化函数连续，从而可以从有翻转映射恢复无翻转。

有限次迭代步后能恢复单射，Chapter4.2 有证明。

Fig11，问了导师说是脚手架方法

Fig12，提出的问题很有趣

如何制造有翻转的网格？

  • 替换两个点
  • 将一部分点旋转一个角度
  • 用 Tutte 参数化计算一个非凸边界的参数化

Optimizing Global Injectivity for Constrained Parameterization (SEA)

与TLC方法同作者。使用该方法可以恢复全局单射。

全局单射和局部单射的定义，组会上讨论了好久。

occupancy 的意思没弄懂，可能是某种拓扑学概念。

处理 Free Boundary 问题。

Isometric Energies for Recovering Injectivity in Constrained Mapping

与 TLC 方法是同一作者。

将原有的两个方法改进到 IsoTLC 和 IsoSEA ，在恢复翻转的同时还能优化等距能量。

成功率并不是100%，有些样例效果还比 TLC 或 SEA 差。

组会上有同学提出为什么先恢复翻转，然后用其他能量优化。导师说可以一步完成肯定是有其意义的。

网格在全局上有交叉 (crossing arm) 时，结果会收缩到一个点。作者也不知道解决方法。

Future Work: 解决上述 overlap 问题。

未想到后续研究的点，该论文搁置。

Simplicial Complex Augmentation Framework for Bijective Maps

脚手架方法，见 3 METHOD段落。算法流程为

  1. 将输入网格添加脚手架 Tutte + [Shewchuk 1996] ( 3D case : TetGen [Si 2015])
  2. 优化扩充后的网格 公式(4)
  3. 更新顶点和脚手架，转至 2 ，直到结果满足条件

优点：生成双射map，可用于生成模型UV贴图

问题：Fig 11 没有优化成功、不适用于 hard constraints

其他研究也可以使用开源代码 Triangle 来进行三角网格生成。如基于 TCB 样条的图像矢量化。
