文献阅读记录（三）

dhui •  2023-10-28  • 0 评论  • 52 阅读

用于记录阅读过的文献，总结文章要点，便于索引。

主要文章采用二级标题，相关文章采用三级标题。

本科毕业设计2022

主要参考了如下文章，实际做毕设的时间只有一个学期。

推导了三角网格上的标量函数的梯度。

使用 libigl 框架，修改相关代码完成目标。

PH-CPF: Planar Hexagonal Meshing using Coordinate Power Fields

导师给的也是最早阅读的，其中涉及到的相关知识点参考文献也基本在这篇文章的引用中找到。

认识到了六边形重网格化问题，以及平面化带来的困难等。这是个需要多个步骤才能完成的工作，每个步骤都需要调研其中的方法，其中有些步骤文章说得很简略，在开源代码中也只提供了一个exe文件。

Dupin Meshing: A Parameterization Approach to Planar Hex-Dominant Meshing

同 PH-CPF ，也提供了六边形重网格化的 pipeline ，但文章更加易懂，先阅读这篇。

学习计算三角形的主曲率和主曲率方向，并编写了 matlab 代码。

负高斯曲率的曲面不适用平面凸六边形。

Dupin Indicatrix 至今不知道翻译。

利用第二基本形式对三角面片分类，用于后续操作。

RoSy = Rotationally Symmetric 方向场

Mixed-Integer Quadrangulation

四边形重网格化。看懂了文章方法。

生成光滑的 cross fields ，然后优化 Mixed-Integer 能量，使用整数优化。（第4,5章）

整数约束：旋转角的整数倍加上整数平移，可以考虑修改到六边形网格上。

Estimating Curvatures and Their Derivatives on Triangle Meshes

计算三角网格第二基本形式的方法。（3.1节）

Globally Optimal Direction Fields

全局方向场生成办法，优化光滑度能量。

可以看作是将 MIQ 方法 n=4 推广到其他n的情形。

N-Symmetry Direction Field Design

粗读，但并未从中看到N-方向场生成的算法

Hexagonal Global Parameterization of Arbitrary Surfaces

其中提到了六边形网格整数坐标的表达式(3)，和自己推导的一致。

有六边形参数化的概念定义。

Directional Field Synthesis, Design, and Processing

方向场综述，对编写毕业论文起到了很大的帮助。

A New Voronoi-Based Surface Reconstruction Algorithm

Crust 算法，用于从点云生成到网格，虽然效果不太好但至少完成了毕设目标。

Practical Mixed-Integer Optimization for Geometry Processing

整数优化的 C++ 开源库，介绍了能解决的问题和用法。
