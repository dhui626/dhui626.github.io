兰斯系列通关感想

dhui •  2023-10-29  • 0 评论  • 3 阅读

虽然中途搁置了好多个月，还是花了一年时间把系列补完了。兰斯系列可以算是a社的代表作了。在做完兰斯10之后TADA退休，做完多娜多娜之后鱼介退社，a社的老牌成员基本散了，现在的a社已经在摆烂。兰斯系列是30年才有一部的作品，也是a社曾经的辉煌。

兰斯系列始于1989年，终于2018年，用十部作品谱写了一位名为兰斯的男人的史诗。就一部一部说起吧。兰斯01，当时a社对这个系列的想法是：要做黄油，正常的男主角很难写出离奇的情节发展所以塑造一个鬼畜的男主角形象，于是就有了这么一部作品。虽然标题是寻找小光，但她直到最后都是一个路人，只是满足了兰斯搞事的欲望和冒险欲。兰斯02，也有重置版，主要介绍了自由都市卡斯塔姆的四魔女，这四人都是后期的重要角色。兰斯03，在第十部之前发行的重置版，也是唯一有语音的一部，同时也是兰斯史诗的开端，有不少说法认为这部是入坑兰斯系列的最佳选择。我也是在这部开始喜欢上兰斯这个角色。兰斯04，去掉重置版最老的一部，主要是03做得太好了，再玩这一部的体验就是断崖式跌落，我直接倍速观看油管的视频通关实况。这部主要和09有关联，介绍了圣魔教团和第一次魔人战争的历史。鬼畜王兰斯，还没玩，这作是TADA在当时写的早期兰斯完结篇，系统很老，但开创了地域压制型SLG的先河。兰斯5D，D是plan D的意思，能理解TADA在当时的迷茫，剧情水准也开始下滑。现在看来确实有必要，前几作都是兰斯的搞事冒险，但这个角色要立起来必须要成长，也需要自食恶果。经历了5D的低谷之后，兰斯06之后就逐渐好起来了。这一部首先开始了多线叙事，不再从兰斯一个人出发，而着重刻画赛斯的各个角色，其中乌鲁泽我很喜欢。兰斯07又名战国兰斯，也是一大名作，很好地呈现了战国时代的魅力，毛利家的三矢训印象深刻，信长的人物刻画也相当不错。兰斯08，刷刷刷游戏，主要是拯救了卡拉族的危机，以及库鲁库成为法王两件事。女儿莉赛特的出场很不错，世界的真相令人震惊。兰斯09，当时玩得热血澎湃，希露被冰封之后兰斯就不再那么孩子气了，成长为了一国的大统领，逐渐往纯爱王发展。兰斯10，系列的完结作，人类齐心协力对抗魔人的侵略，6代开始的多线叙事已经十分成熟，无论是在音乐、游戏性、剧情深度上都达到了巅峰。至此，兰斯漫长的旅程迎来结束。

以上是剧情的评价，关于游戏性，a社是肯下功夫钻研的。兰斯每一部都换一个游戏引擎，也创新一种游戏玩法，这也导致每一部都可以当做新游戏玩。不过有几部创新是失败的，有很多反人类的设计，比如兰斯5D的剧情推进靠掷骰子，运气不好就game over，兰斯06的靠地图上捡玉推进剧情，如果不开修改器会多浪费一倍的时间。不过也有好玩的，特别是战国兰斯这个系统，太有趣了。兰斯10的游戏性也很强，击杀魔人会提高难度，一个回合只有一次作战，需要讲究策略，一不小心就会被灭国。

下面单就兰斯10作个评价吧。兰斯10的游戏流程是真的长，首先体现在剧情流程上。游戏分为第一部(相当于Normal End)和第二部(相当于Ture End)。第一部的可操作性较强，玩家控制主角团击退魔人，解放四国，并解决美树的魔王化，达成正史结局。第二部实现了剧情的升华，故事发生在十五年后的世界，很好地刻画出了年代感，一些前作角色的变化很有看点，令人感慨。且第二部将第一部的几个伏笔都填上了，且结局堪称完美，兰斯的旅途真正意义上结束了。除掉剧情流程之外，也有不少有趣的部分。兰斯10共有700万字的文本，红楼梦都才100万字呢。其中很大一部分都在角色的餐券剧情上，这是很不容易的。兰斯系列出场的角色有400+，几乎每一个角色都有个人剧情，很难想象TADA花了多少心血在上面。餐券剧情越看越觉得厉害，人物关系错综复杂最终都在兰斯这里形成闭环，越来越觉得兰斯的世界观是生动而又有趣。

前段时间还经常闲着没事打开兰斯10看几个餐券剧情，现在差不多可以封盘了，游戏时长190h，晚安，兰斯。
