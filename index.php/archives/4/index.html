文献阅读记录（一）

dhui •  2023-10-28  • 0 评论  • 56 阅读

用于记录阅读过的文献，总结文章要点，便于索引。

主要文章采用二级标题，相关文章采用三级标题。

计算机图形学2021

接触 CG 的开始！涉及到的一些文章都是一些经典文章，适合仔细阅读几次。

Poisson Image Editing

泊松图像融合，网上能找到很多中文材料。

学习了信息整合的方法，尽管仅限于中文资料。

Seam carving for content-aware image resizing

方法很简单，但能想到属实不易。

参考资料： Seam Carving 图片缩放

调研的图像特征提取方法： 四种比较简单的图像显著性区域特征提取方法原理及实现

Parametrization and smooth approximation of surface triangulations

Tutte 参数化经典文章。

Bonus：用 Floater 权重实现保形参数化，参考资料 三角网格曲面参数化

改进遇到了中文资料找不到的难题，还好 Google 到了上述博客，解决了问题。

Laplacian Surface Editing

认识到了 MATLAB 和 C++ 混合编程的厉害。

还能找到中文参考资料：用数学编辑3D模型（一）

Bonus：提供选项可用于对 handle 的 Laplace 坐标进行旋转操作。参考资料：四元数和旋转

但是对四元数不熟悉，实际交上的作业未完成上述目标。

2023.10.13 再看：该文章也是从图像处理推广到网格的方法，motivation 很 trivial.

As-Rigid-As-Possible Shape Interpolation

ARAP 形状插值。插值映射的 Jacobi 矩阵。

能找到的中文资料越来越少，最后妥协读英文文献了。参考

Bonus：解决旋转一致性问题

阅读文献 Rigid Shape Interpolation Using Normal Equations 第七节解决。

Fast Simulation of Mass-Spring Systems

质点弹簧系统仿真，实际阅读是刘利刚老师的ppt并未阅读论文。

自行实现的 C++ 代码运行结果有点问题。

Taichi: a language for high-performance computation on spatially sparse data structures

Taichi (太极) 框架，实际使用框架修改参数进行了几种物理模拟，并未阅读论文。

Ray tracing in one weekend/The next week/The rest of your life

计算机图形学前沿的自选课题，旨在用 C++ 手撕光线追踪算法。

第一本书进行手把手操作复现了代码，但后两本仅停留在阅读阶段。
