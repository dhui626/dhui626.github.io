Heaven Burns Red 剧情感想

Heaven Burns Red 第二章通过。

220317_0354

只能说不枉我熬夜看剧情到现在，麻子的剧本水平还是一流的。

虽然第二章的剧情发展从本月的活动标题里就能猜出个大半，但我还是折服于麻子的文笔，哭成泪人。苍井拥有过目不忘的能力，也是29班的某次行动中唯一的幸存者，从那以后就一蹶不振，却当上了31B的队长。31B本是支离破碎的队伍，但实际上大家是为了不让队长苍井再度前往战场，故意不合作。月哥哥为了鼓舞苍井振作起来，和她谈心解开了心结，也让31B团结了起来，但是最终。。让我无比佩服的是本作世界观架构的揭露，当あおい化身成ナービィ出现在月歌的身边时，之前的好多疑问都豁然开朗了，谁会知道在公园里活蹦乱跳的可爱生物都是故人的亡魂，他们不会说话，人们也无法与之交流，但这些谜之生物却在守望着活着的人，两者的隔阂正是生死之隔。第二章通过后再来看前面的剧情，只能说是五味杂陈。不管怎么说，我认为这次麻子的剧本是很成功的，我就吃这一套。

Heaven Burns Red 第三章通过。

220430_0326

个人认为第三章各个方面都弱于第二章，远没有第二章给我带来的震撼大。看了今天开的第三章补充剧情再评价吧。就剧情而言还在展开期，不过目前公开的部分就到这里为止。

然后不得不吐槽第三章关底boss，这是我打得最折磨的一个，第一天用6SS最高战力打才勉强磨过一阶段，第二天用b站的推荐阵容打，因为要趁boss蓄力的间隙上buff，两个S角色被挨到一下就寄了，重试了好多次，第三天继续反复试错和记录boss蓄力的回合数，还是打不过。今天第四天直接把那两个S的角色换掉，只按攻略说的把精神堆到290以上，终于勉强打过二阶段，并留住了国见的一个S技，得以抗住boss三阶段三次高伤害。

虽然打的时候一直在骂，打过之后看了剧情我反而觉得这个boss设计是很精妙的。按照剧情展开这里确实需要设计一个非常难打，非常难缠，还打不死的boss，从而让玩家抛出这个家伙为什么打不死的疑问，后面的剧情发展才得以顺水推舟。不过就手游而言是很失败的，堆精神数值到290以上抗伤害是个试探box深度的行为，没6个SS还真打不过，不对，就算有再多的SS不研究游戏机制不可能打过。

Heaven Burns Red 第三章活动剧情结束。

220503_0107

我认为这个篇章才是本体，对第三章唐突结束的结尾是一个很好的补充。只能说还是key社的味道，麻子总会在细节上触发玩家的泪腺，我估计这辈子都会是键厨吧。Visual Arts什么时候把专辑吐出来啊啊。

红烧天堂六月活动剧情通过。

220618_0159

这个月的活动剧情看起来很水，但很有可能是世界观的伏笔。 以下可能有剧透。

红烧天堂第三章就揭露了部分世界观了，为什么人类死后会转生史莱姆，司令官把这些消息列为绝密档案仿佛是想在隐藏什么，这期活动里司令官也说到：也有其他捕获キャンサー并进行秘密研究的组织啊，什么反派言论。本月幻境的boss追加几个没见过的，很可能是第四章boss，大胆猜测下个月会开第四章，月歌又能将自身的起源解明到哪种程度呢，很是期待。

Heaven Burns Red 七月活动剧情通过。

220716_0028

这个月忙着玩megido72一度想搁置这款游戏，事实上确实咸鱼了不少，每天体力堆到上限也不管了，就登录收个周回。不过这个月的活动剧情不负期望。

这次主要展开讲了31E的故事，一支神奇的队伍，队员是六姐妹。作为队长的姐姐在一次行动中不幸失忆，不记得与队员的关系了。所幸通过天音的帮助暂时恢复了部分记忆，身为年纪最大的姐姐最先感受到的是压力，这点很有趣。最后姐姐在一次哨戒行动中意外到达了年幼时去过的地方，六姐妹的过去被揭示，终于完成了过去未完成的绘马。作为姐姐一直以来承担的压力与责任得到了理解与承担，麻子真是个温柔的人啊。

Heaven Burns Red 八月活动剧情通过。

220819_0007

表面泳装回实际上是恐怖向，然后一转麻子风格。这次的活动剧情也是第二章剧情的补充，不过之前的いちご视角更现实一些，这回是月歌视角。这个月开放的第四章前半也是一大惊喜，不过还是打算搁置到后半开放再玩。

红烧天堂每月的活动剧情都能让我哭一次，每次都对麻子佩服得五体投地。ユキ：“睡昏头了吧，现在这世道还能去游乐园玩吗” 月歌：“玩得很开心哦，和アオイ一起”

Heaven Burns Red 九月活动剧情通过。

220908_0124

以拍摄ヒーロー映画为主线引出キャロル的过去，算是她的个人剧情吧。虽然麻子有些放水，依然是个感人的故事。キャロル和姐姐是相互扶持的关系，キャロル从小就憧憬着担当英雄动画导演的姐姐，姐姐在迷茫的时候也因为妹妹的憧憬才没有迷失方向。看完姐姐留下的动画之后，キャロル第一次摘下了眼镜：这是最棒的ヒーロー映画！

Heaven Burns Red 十月活动剧情通过。

221006_0154

白河ユイナ个人回，发现她也有和yuki一样当吐槽役的潜质。她曾数次得到“天启”，虽然不太相信但也逐渐与月歌接触，直到特别哨戒作战（主线第二章结束时间点）看到光柱的那一刻突然醒悟，于是为守护月歌的战斗默默帮她们扫除障碍。从那以后ユイナ开始相信天启，逐渐知晓了自己将来将要背负的命运，并告诫月歌一定要坚强地战斗下去。第二章的side story，立意联想到了jo5的黄金精神，虽然知道自己的命运却还是毅然坚持着走下去。泪目，不是在看剧情的时候，而是在挂上周回关了游戏后。

Heaven Burns Red 十一月活动剧情通过。

221105_0220

柊木梢个人回，补全她过去经历的同时也揭开了眼罩的秘密。月歌哪来的通灵能力，突然奇幻展开？梢也终于和儿时的伙伴达成和解，她摘下眼罩真的好帅气！通关之后整了一发十连果然沉了，飞机社真有你的啊。

Heaven Burns Red 十二月活动剧情通过。

221212_0217

夏目祈个人回，探讨了挥剑的意义。处刑人是什么年代的，昭和年代也没有了吧。她的剑是断罪之剑，也是慈悲之剑，温柔的剑。她虽在セラフ剑术大赛中败北，却也领悟到了挥剑的目的，并继续精进。

怀疑放水了，可能在憋下个月的企划或者主线，但暂时没时间玩第四章主线。

Heaven Burns Red 一月活动剧情通过。

230121_1203

麻子又被网友气的删推了。就之前的活动剧情来看，这次活像一个同人故事，无人岛上与人鱼的邂逅，说这梦幻也好吧，但是人鱼是什么物种，主线里从来没提到过啊，说麻子磕了什么东西写出来的也不为过吧。确实能感受到剧情质量的下滑。

key又开始搞小游戏了，这玩意就一个伊苏8，跑3d地图捡材料也太无聊了。day3做了个料理，防御大减，打boss差点寄了。

顺带一提，最近开的扫荡机制很不错，打一次幻境花一管体力，几个主力瞬间110满级了。反正体力已经99上限了不值钱，且根本懒得肝。还是メギド72好玩。

Heaven Burns Red 二月活动剧情（Angel Beats联动）结束。

230228_0131

飞机社已经把手游玩明白了，一周年繁中更新以及限定Angel Beats联动，大批玩家涌入，策划躺着赚米。顺便用steam版测试了繁中版，感觉不错，不过手机版全语音看日文也不影响理解就是了。

可是三位就这样入队了？不会在主线登场吧。不管怎样最近没啥过主线的欲望。没玩过ab，入江是ab里的人物还是原创人物呢。故事展开还是在情理之中的，对于入江的死最为排斥的人反而是最关心她的人，以及人总要前进，为此总有一日要放弃过往，麻子也开始喜欢写大众化的剧本了。

Heaven Burns Red 活动剧情「君に読む憧れ」结束。

230411_2147

上个月活动剧情竟然错过了，看来这游戏对我来说的优先级越来越低了。也许可以开始摸第四章主线了？4.28日解锁主线后半。

麻子写活动剧情质量有些起伏不定，不过这期质量还算不错，和刚开服时候的マリ活动剧情连上了。主要揭示了神崎成为忍者的理由。神崎幼年被绑架被不知名忍者所救，从此开始了忍者的修行，以为合格的忍者从来不会让别人知道自己的身份，但神崎却到处炫耀，目的也是希望自己的修行成果能传到曾经救过她的忍者耳中。麻里继承家业失败，却意外地收获了追求者：那家伙已经是一个比我出色的忍者了。神崎也救下了一位立志成为忍者的小孩，小孩和幼年神崎十分相似，忍者的事业想必也得到了传承。

Heaven Burns Red 活动剧情「大島屋物語」结束。

230414_2355

上个月漏掉的，大岛家六姐妹在活动剧情里出场这么频繁，且过去已经基本补完了，估计主线就见得比较少了。

上一回大岛家活动剧情只讲了六人之间的羁绊，还没有触及核心。这回叙述了大岛家的过去。兴致勃勃想要继承家业的六宇亚在开店模拟中逐渐想起了过去，父母的事业明面上是商店，但背后是黑市武器商，因此六宇亚在幼年时被绑架，遭受了一段悲惨的经历。而知道过去的六宇亚在姐妹的鼓舞下重振信心，看到开店的成员从起初的2人到最终的6姐妹全员以及得到了其他部队的帮助，还是十分感动的一件事。

Heaven Burns Red 主线第四章前篇结束。

230424_2134

现在玩可以赶上4.28后篇解放，连贯地体验故事，基于这个想法才来玩的。等级上限上调至110之后轻松不少，跑图也不用担心回复不过来。雷队成型之后第四章这个轻轻松松，没有感受到当时全队90级硬抗第三章boss的折磨。

第四章最有感触的是前三天的剧情，全员对纳比的真相从震惊到理解与接受，月歌演奏主题曲吉他版真好听，但只有megumi一直没和真相达成和解。这部分把除了月歌和yuki以外四位的过去都揭示了，两位官方cp的爆点应该在更后面。结尾我和玩了第三章一样想打出一串问号，希望后篇能把故事讲好吧。

后篇双线叙事可以期待一下，且增加了一些游戏形式，3D走路遇敌真是够了，还有头目的血条变成了四管，希望不要太坐牢，玩起来和前篇一样舒服吧。

Heaven Burns Red 断章 遥远的海色 结束。

230425_1320

第四章前篇的补充剧情，为解决色葉陷入自我认知危机而看不见颜色的问题，黑化可怜酱大活跃，之后色葉找到了过去被母亲抛弃的真相，原来母亲也曾是一位研究员，和东城类似，为了人类的伟业不得已才将她送到亲戚那里寄养。也许可怜酱在主线后篇面对惠惠也会这样。后篇要开始展开世界观了，已经等不及喽。断章里有个CG貌似是月歌对过去的回忆，主线都没有提到过，大概暂时还不会展开说吧。

Heaven Burns Red 第四章后篇结束。

230509_1636

4.28解放的内容，差不多花了十天，还是挺感慨的。就难度而言，和第四章前篇相似或难一些。关底boss战比较简单，前半不需要太多战术就能击破4条dp槽，后半知道继承前一个阶段攒的od不要用掉就行。最难的是关底boss前的小怪，10万dp，60万hp，且破盾后8回合会回满dp。该小怪需要使用两个队伍打两次，角色不能有重复，昨晚研究了挺久，最后凑了火队和冰队打，正好关底boss战也用上了。打通后觉得这两个小怪的设计是合理的，因为关底boss的设定是越来越强，四阶段的难度挺大。小怪的作用相当于是作为一个门槛，大大提高玩家关底boss通过率，能够更好地享受剧情。

四章后篇应该是至今为止的最强主线了。双线叙事，尽管只有14天但是文本量很丰富。一条线路主要讲了megumi找回自我的过程之后说，另一条线路讲月歌队伍在缺少队员的情况下的挣扎与自我突破。最后一天的演出太帅了，满足了期待，惠，欢迎回来！新篇章也是备受期待，这一章挖掘了31A四人的过去，终于是时候讲述月歌的过去了吗。

惠侧线路让人耳目一新，世界观大展开，失去斗志的她开始和避难所的人类一起生活，寻找活着的意义。一大亮点，钓鱼，钓鱼是有纪家的食物和收入来源，做成小游戏很合理，钓鱼成果会影响有纪家的晚饭丰盛程度，且后期公正币的主要赚取方式就是这个。一大亮点，唱片机，可以点到key社之前作品的曲子，如夏影等，有时候点播了歌曲之后都不想接任务了，只想静静听歌。一大亮点，富有特色的人物特征，如冰激凌屋的相扑选手体型和口癖，猎头屋小孩对主角的boy蔑称，明明是酒馆却用居酒屋方式接客的店员，看似不均衡但各司其职的人类社会结构等。一大亮点，有纪的过去，虽然暗示给得很明显很早就猜到了，但揭露真相的演出还是让我泪目了，这种单方面知道真相却难以倾诉的情感太让人难过了。这些亮点结合起来，足够让玩家也能领会到避难所艰苦却又十分满足的生活，以及失去它的难过。惠最后用超能力解决危机太帅气了！！

Heaven Burns Red 活动剧情「罪と罰と愛と」结束。

230607_2328

终于开始讲31X的故事了。作为一个各个国家挑选出的精英组成的团队，却是个人战斗力高但团队战斗力最低的团队，到第四章主线的时候也为她们捏了一把汗。揭露了夏洛的过去经历，屡次经历战场活下来的少女，被说是拥有死亡的诅咒的少女，见证了太多同伴的死也成为了她的心理创伤。关于这不幸中的万幸，司令官说这和她体内的星癌体碎片有关。这次遇上了同样的星癌体，过去的经历险些要重现，好在31X是一支强大且团结的部队，成功救场，这应该会成为31X的战斗风格从个人战向团体战转变的契机吧。

Heaven Burns Red 七月活动剧情「Peace of Cradle」结束。

230711_1052

意外的质量很高，救下的少女在day2夜晚身份揭露时的反转很耐人寻味，意料之外情理之中，可算解释了一夜在设施里活动而理沙却从没有被司令官约谈的原因，以及举高高原来是这样的恶趣味。且通过这次剧情揭露了理沙的过去，结尾不用猜就是麻子写的（参考AIR）。这次的主题曲超好听！

Heaven Burns Red 八月活动剧情通过「きみはこの夏のFairy、ぼくはその姿を瞳の奥にRec.」

230814_0017

好长的标题，繁中的翻译我还看不懂，以为这个月没有新的曲子了结果最后放了上来。我觉得麻子已经顶不住了，写不出新东西了，每个月进行一次这样的商业剧本创作真不容易。成功的视角欺骗，活动boss跟捏死一只蚂蚁一样简单，实际上的真boss是flathand。近期的活动剧情已经骗不到我的眼泪了，就算它BGM放个夏气球也不行了，是我年纪大了泪点提高了吗，还是在通了第四章后篇之后采用了更高的视点看待第三章这个事情了呢。

Heaven Burns Red 九月活动剧情「小さき帝と穏やかな朝食」结束。

230914_0118

补充了柳和丸山家的故事，两人的羁绊从幼年时就开始了，柳作为执事经历了家族变迁但也在默默守护和支持丸山，希望丸山自立自强走上自己想走的道路，这也是母亲的愿望。幼年的两位好可爱！最近hbr上线好麻烦，直连卡loading，只有steam版能玩。不过也佛系了，偶尔上线。但是每期新的音乐总是不会让我失望。

Heaven Burns Red 十月活动剧情「美人温泉物語」结束。

231010_1323

音乐好听！虽说是温泉慰劳旅行但没有温泉，实际上是被骗过去进行的作战任务，借此揭露了二以奈和浅见教官的过去。二以奈为了保护脸蛋不受伤，作战总是退居二位，过于顾虑别人，但这次经历让她做出了改变，保护同伴的意志更加坚定了。浅见教官在此处失去了最亲的人且丧失了战斗的动力，从此成为教官，但最后召唤出炽天使和二以奈并肩战斗好酷！最终终于说出了那句：いい湯だな。

玩的同时也在思考，为什么麻枝能在hbr上取得成功。在之前的galgame里，玩家需要阅读30h的文本来收获1h的感动，而在现在碎片化信息盛行的时代，大部分玩家都希望在短暂的时间内收获同等的感动，而hbr做到了这点。虽然这确实是迎合市场做出的最佳手段，但收获的感动也是暂时的，不像lb、rw等作品让人回味无穷，咱还是希望能多看到一些长篇作品的涌现啊。
